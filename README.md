## Setup
Before you run the server please use these commands:

```
$ pip install Flask Flask-Cors Flask-PyMongo

```

To run this project, first run the server:

```
$ cd backend/src/
$ py App.py

```
then run client side server:
Open new terminal, then:
```
$ cd frontend/src
$ npm install
$ npm start
```
